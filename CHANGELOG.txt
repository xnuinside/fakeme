**v0.1.0 (in process)**
1. Added simple-ddl-parser as dependency to extract information from DDL, code refactoring
2. Added code changes to support Python 3.8 and upper (relative to changes in python multiprocessing module)
3. Added tests runner on GitHub
4. Autoaliasing fixed
5. Added a bunch of unit tests
6. Now you can pass tables definitions not only as list, but also as Table class:
```python
from fakeme import Table

Table(name='table_name_example', schema='path/to/schema.json')

# or 
user_schema = [{'name': 'id'},
          {'name': 'title'},
          {'name': 'rights', 'type': 'list', 'alias': 'right_id'},
          {'name': 'description'}]
Table(name='table_name_example', schema=user_schema)

```
